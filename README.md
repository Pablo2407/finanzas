# 💰 Sistema de Gestión Financiera con Flask
**Desarrollador:** Juan Pablo | Python Intermedio

Esta es una aplicación web robusta construida con el micro-framework **Flask**. Permite a los usuarios gestionar su economía personal con herramientas de nivel profesional.

### 🛠️ Stack Tecnológico
- **Backend:** Python & Flask.
- **Frontend:** HTML5, CSS3 (JinJa2 templates).
- **Reportes:** Generación dinámica de archivos **PDF** y **Excel**.
- **Base de Datos:** SQLAlchemy.

### 🚀 Funcionalidades
- **Arquitectura Web:** Manejo de rutas, plantillas dinámicas y peticiones HTTP.
- **Exportación de Datos:** Procesamiento de información en servidor para descarga de documentos (PDF/Excel).
- **Lógica Financiera:** Algoritmos en Python para el balance de saldos y categorías.
